🎯 Demo Workflow
1
Login via Auth
Go to Authentication → use Quick Fill, then Login
2
Add a Book
Go to Books → click "Add Book", fill the form
3
Create Member Profile
Go to Members → Add Member (use User ID from Auth)
4
Create a Loan
Go to Loans → New Loan (copy Book ID & Member ID)
5
Return the Book
Go to Loans → My Loans → click Return
6
Check Fines & Monitor
Fines section for overdue charges, Monitor for health
ℹ️ Architecture
All requests go through the API Gateway at :8080 which validates JWT and forwards X-User-ID + X-User-Role to downstream services.

Services: Auth (:8001) · Book (:8002) · Member (:8003) · Loan (:8004) · Fine (:8005)
🔑 Test Credentials
Register a user first if no accounts exist:
Role admin — full access to all endpoints
Role librarian — manage books, members, loans
Role member — borrow books, view own data
